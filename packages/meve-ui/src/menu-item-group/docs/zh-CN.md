## API

### 属性

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `name` | 菜单组名，唯一标识 | _string \| number_ | **-** |
| `label` | 菜单组文本 | _string \| number \| Function_ | **-** |
| `icon` | 菜单组图标 | _string \| IconProps \| Function_ | **-** |
| `disabled` | 是否禁用菜单组 | _boolean_ | **false** |

### 插槽

| 插槽名 | 说明 | 参数 |
| --- | --- | --- |
| `label` | 菜单组文本 | **-** |
| `icon` | 菜单组图标 | **-** |
